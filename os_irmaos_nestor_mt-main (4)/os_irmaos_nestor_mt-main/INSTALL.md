# Instruções de Instalação - Extensão OS IRMÃOS Anti-Bullying

## 📱 Para Dispositivos Móveis

### Android (Chrome)
1. **Baixe o Chrome** (se não tiver instalado)
2. **Acesse** chrome://extensions/ no navegador
3. **Ative** o "Modo do desenvolvedor"
4. **Instale** a extensão seguindo as instruções para desktop
5. **Configure** as palavras através do popup da extensão

### iOS (Safari)
⚠️ **Nota**: Safari no iOS tem limitações para extensões de terceiros. Recomendamos usar Chrome ou Firefox.

## 💻 Para Desktop

### Windows
1. **Baixe** a pasta da extensão
2. **Descompacte** se necessário
3. **Abra** o Chrome ou Firefox
4. **Siga** as instruções específicas do navegador abaixo

### macOS
1. **Baixe** a pasta da extensão
2. **Descompacte** o arquivo se necessário
3. **Abra** o Chrome ou Firefox
4. **Siga** as instruções específicas do navegador abaixo

### Linux
1. **Baixe** a pasta da extensão
2. **Extraia** o arquivo se necessário
3. **Abra** o Chrome ou Firefox
4. **Siga** as instruções específicas do navegador abaixo

## 🌐 Instruções por Navegador

### Google Chrome / Chromium / Edge

1. **Abra o navegador** e digite na barra de endereços:
   ```
   chrome://extensions/
   ```

2. **Ative o "Modo do desenvolvedor"** no canto superior direito

3. **Clique em "Carregar extensão sem compactação"**

4. **Navegue** até a pasta `anti-bullying-extension`

5. **Selecione a pasta** e clique em "Selecionar pasta"

6. A extensão aparecerá na lista e na barra de ferramentas

### Mozilla Firefox

1. **Abra o Firefox** e digite na barra de endereços:
   ```
   about:debugging
   ```

2. **Clique em "Este Firefox"** na barra lateral esquerda

3. **Clique em "Carregar extensão temporária"**

4. **Navegue** até a pasta `anti-bullying-extension`

5. **Selecione o arquivo** `manifest.json`

6. A extensão será carregada temporariamente

### Microsoft Edge

1. **Abra o Edge** e digite na barra de endereços:
   ```
   edge://extensions/
   ```

2. **Ative** o "Modo de desenvolvedor" no painel esquerdo

3. **Clique em "Carregar extensão descompactada"**

4. **Selecione** a pasta `anti-bullying-extension`

5. A extensão será instalada

## 🔧 Configuração Inicial

### Primeiro Acesso

1. **Clique** no ícone da extensão (dois meninos de mãos dadas)
2. **Digite a senha**: `osirmaos`
3. **Clique em "Entrar"**

### Personalizando as Palavras

1. **Acesse** o painel de gerenciamento (após login)
2. **Adicione** novas palavras no campo "Nova palavra para bloquear"
3. **Remova** palavras desnecessárias clicando em "Remover"
4. **Salve** as alterações automaticamente

## 🎯 Testando a Extensão

### Teste Básico

1. **Abra** qualquer site (ex: Google, Facebook, Twitter)
2. **Digite** uma palavra da lista de bloqueio em um campo de texto
3. **Observe** se a palavra é substituída por símbolos (●●●)
4. **Verifique** se o popup educativo aparece

### Teste de Palavras Padrão

Experimente digitar estas palavras em qualquer site:
- "idiota"
- "burro" 
- "estúpido"
- "feio"

## 🛠️ Solução de Problemas

### Extensão Não Aparece

1. **Verifique** se o "Modo do desenvolvedor" está ativado
2. **Recarregue** a página de extensões (F5)
3. **Verifique** se todos os arquivos estão na pasta correta

### Senha Não Funciona

1. **Certifique-se** de digitar: `osirmaos` (tudo minúsculo)
2. **Limpe** o campo de senha e digite novamente
3. **Verifique** se não há espaços antes ou depois

### Palavras Não São Bloqueadas

1. **Recarregue** a página onde está testando
2. **Verifique** se a palavra está na lista de bloqueio
3. **Confirme** se a extensão está ativada

### Modal Não Aparece

1. **Desative** bloqueadores de popup
2. **Recarregue** a extensão em chrome://extensions/
3. **Teste** em modo anônimo/privativo
